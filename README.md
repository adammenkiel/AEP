## What is this?
That is an experimental mini-framework that implements several variants of predictive models based on generating random expressions and alternatively operations at high-scoring ones. At this moment it work with binary classification but it's will be updated soon!

## Examples
Examples can be found in `src/main`.

## Tip:
You can use `src/main/java/pl/publicprojects/predictor/basic/RefactorClass.java` to merge result group of expressions into one expression just put your model output into RefactorClass input and write `end` then you need to change variables named ``$0$, $1$ ... $n$`` into $x$, $y$, $z$ ... and so on. Finally you can put your query data into these variables and you will get answer by checking condition `expression > 0`
