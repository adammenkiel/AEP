## What is this?
That is an experimental mini-framework that implements several variants of predictive models based on generating random expressions and alternatively operations at high-scoring ones

## Examples
Examples can be found in `src/main`.

## Tip:
You can use `src/main/java/pl/publicprojects/predictor/basic/RefactorClass.java` to merge result group of expressions into one expression.
