## Reporting a Vulnerability

If you want to report any vulnerability, please do it responsibly.

You can report by following ways:
- Email: akmenkiel@gmail.com
- Github Issues: https://github.com/adammenkiel/AEP/issues

Please include as much detail as possible, such as:

- steps to reproduce the issue
- affected version(s)
- potential impact

We will review your report as soon as possible and respond with further information. You can expect an initial response within a reasonable timeframe.
